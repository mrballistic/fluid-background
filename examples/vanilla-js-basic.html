Basic Usage

This example demonstrates the basic usage of SplashCursor in vanilla JavaScript:

// Import the library
import { createSplashCursor } from 'fluid-background/splash-cursor';

// Create and start the splash cursor
const splashCursor = createSplashCursor({
    intensity: 0.8,
    particleCount: 150,
    colors: {
        mode: 'rainbow',
        saturation: 80,
        lightness: 60
    }
});

splashCursor.start();
